Información del sitio

Abril de 2016


Bienvenidos! a éste Blog que fue realizado desde cero con HTML5, CSS3 y JavasScript utilizando Bootstrap un framework de diseño web responsive (adaptable a las pantallas de todos los dispositivos), próximamente se le agregara una conexión con base de datos MYSQL y un lenguaje de servidor PHP para que se puedan guardar los datos dinámicamente y así las personas podrán comentar en las publicaciones, realizar búsquedas, entre otras cosas con el fin de completar el desarrollo de este sitio web. Pueden ver o descargar el código fuente de éste blog a través de GitHub
