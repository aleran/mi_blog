Informacion del sitio

07 de abtril de 2016

Bienvenidos a éste blog de tecnología y desarrollo web, el cual creé con el fin de realizar un proyecto donde yo pueda aplicar mis conocimientos y poder trasmitirlos a cualquier persona, asi como yo he aprendido gracias otras páginas o blogs de programación donde se comparte información, videos de cursos, tutoriales y experiencias en el ámbito del desarrollo de software.

El Blog fue realizado desde cero con HTML5, CSS3 y JavasScript utilizando BootsTrap un framework de desarrollo web responsive (adaptable a las pantallas de todos los dispositivos), próximamente se le agregara una conexión con base de datos MYSQL y un lenguaje de servidor PHP para que se puedan guardar los datos dinámicamente y así las personas podrán comentar en las publicaciones, realizar búsquedas, entre otras cosas con el fin de completar el desarrollo de este sitio web.
